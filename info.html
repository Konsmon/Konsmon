What is KONSMON?

Konsmon.space is a DIY chatting website. Users can create their own private or public servers with text and voice channels. There is also a possibility to stream your screen to other users. You might wonder: why create such a website when apps like Discord or TeamSpeak exist? The answer is simple and straightforward. It was created with the idea of making a free and unregulated alternative for small and specific purposes, e.g., university groups, etc.

If you are interested in more technical details, you can check out the GitHub repository: github.com/Konsmon/Konsmon

The website was created with love for freedom of speech by: Frans and JulekJestem.


Here is an easy step-by-step tutorial on how to use the website:
  1. Go to the main page and click "Create Server".
  2. Choose a name and password for your server (password is optional).
  3. Click "Create" and wait a moment.
  4. And that's all! Now, you can customize your server and create more voice or text chats.
  5. REMEMBER! The website is still being developed; there might be plenty of bugs and security flaws.
